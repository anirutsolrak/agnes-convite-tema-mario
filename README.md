# 🎮 Convite Animado — Agnes Faz 6 Anos!

Convite digital interativo estilo Super Mario para a festa de **Agnes**, **6 anos**, no **Playcenter Family Aricanduva** — **27 de junho às 12h**.

---

## 📂 Estrutura do projeto

```
invite-agnes/
├── index.html          ← página principal
├── css/
│   └── style.css       ← todos os estilos e animações
├── js/
│   └── main.js         ← lógica, interações e sons
└── assets/
    ├── invite_card.jpg ← imagem do convite (Agnes)
    ├── coin.png        ← moeda animada
    └── characters.png  ← Mario & Luigi
```

---

## 🚀 Deploy no GitHub Pages

1. Crie um repositório no GitHub (ex: `invite-agnes`)
2. Suba todos os arquivos mantendo a estrutura de pastas acima
3. Vá em **Settings → Pages → Source → Deploy from branch → main → / (root)**
4. Aguarde ~1 min e acesse `https://seu-usuario.github.io/invite-agnes/`

---

## 🎮 Como funciona

| Tela | O que acontece |
|------|----------------|
| **Start** | Mario & Luigi flutuam, moeda gira em 3D, letras "APERTE O START" caem uma a uma |
| **Click** | Tela some com efeito zoom, confete estoura, convite revela com animação elástica |
| **Easter egg** | Clique 3× na moeda → Mario corre pela tela 🏃 |
